///// 1. o que é react? O React é uma biblioteca front-end JavaScript de código aberto com foco em criar interfaces de usuário em páginas web. ///// por que é chamado de biblioteca? É chamado de biblioteca, porque não obriga estruturas e regras como frameworks, permitindo usar o que precisamos minimamente para a criação de uma interface com mais flexibilidade de código e visual, com foco em criação de interfaces para WEB, MOBILE (android & IOS).//
